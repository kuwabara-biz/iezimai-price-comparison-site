# iezimai-price-comparison-site
実家じまいサイトのプロトタイプを作成

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## 開発の始め方 (Getting Started)

開発サーバーを立ち上げるには、以下のコマンドを実行してください：

```bash
npm run dev
# または
yarn dev
# または
pnpm dev
# または
bun dev
```

コマンド実行後、ブラウザで [http://localhost:3000](http://localhost:3000) を開くと結果を確認できます。

ページの内容を編集するには `app/page.tsx` を修正してください。ファイルを保存すると自動的に更新が反映されます。

## 開発サーバーの停止方法

開発サーバーを停止するには、ターミナルで以下のキーを押してください：

- `Ctrl + C`
